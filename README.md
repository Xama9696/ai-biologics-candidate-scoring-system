# AI-Based Biologics Candidate Scoring System

A machine learning project to evaluate and rank biologic drug candidates for early-stage wet-lab experimentation.

## Objective
Shortlist high-potential biologic candidates using scientific developability metrics.

## Features
- Synthetic biologics dataset generation
- Candidate scoring framework
- Classification model for shortlisting
- Candidate ranking dashboard
- Feature importance analysis

## Input Variables
- Stability
- Binding Affinity
- Solubility
- Toxicity
- Expression Yield
- Immunogenicity Risk

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Power BI

## Outcomes
- Reduced manual screening effort
- Faster candidate prioritization
- Data-driven R&D decision support
